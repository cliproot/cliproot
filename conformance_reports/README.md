# Conformance Reports

This directory stores conformance result envelopes generated from canonical vectors.

## Layout

- `v0.3/<implementation>/<YYYY-MM-DD>.json`

## Notes

- Files here are public implementation evidence artifacts, not certification records.
- `v0.3/spec-baseline/2026-03-03.json` is a seed artifact to demonstrate envelope format and matrix generation.
- Replace seed/baseline artifacts with real runtime execution reports as implementations are added.
